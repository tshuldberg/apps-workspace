# MyLife Hub

MyLife is a unified, privacy-first personal app suite for iOS, Android, and Web. All data stays on-device by default. Users enable 41 registered modules from a hub dashboard, powered by a single SQLite database with prefixed tables per module.

## Stack

- **Mobile:** Expo (React Native) for iOS and Android
- **Web:** Next.js 15 (App Router)
- **Database:** SQLite (expo-sqlite mobile, better-sqlite3 web)
- **Monorepo:** Turborepo + pnpm
- **Language:** TypeScript
