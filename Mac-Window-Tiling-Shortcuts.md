# Mac Window Tiling & Arrangement Shortcuts

All shortcuts use the **Control + Globe (Fn)** modifier combo.

---

## Core Window Controls

| Shortcut | Action |
|----------|--------|
| ⌃🌐 F | **Fill** — maximize window on current desktop |
| ⌃🌐 C | **Center** — center window on screen |
| ⌃🌐 R | **Return** — restore window to previous size |

> **Fill vs Full Screen:** Fill maximizes within the current desktop. Full Screen (green button or `View → Enter Full Screen`) creates a separate Mission Control space.

> **Fill vs Zoom:** Zoom resizes a window to fit its content optimally. Fill always maximizes to full desktop size.

---

## Screen Halves

| Shortcut | Action |
|----------|--------|
| ⌃🌐 ← | Tile to **left half** |
| ⌃🌐 → | Tile to **right half** |
| ⌃🌐 ↑ | Tile to **top half** |
| ⌃🌐 ↓ | Tile to **bottom half** |

---

## Two-Window Splits (add Shift)

Tiles the **top two windows** simultaneously with one command.

| Shortcut | Action |
|----------|--------|
| ⌃🌐⇧ ← | Split top two windows into left/right halves |
| ⌃🌐⇧ → | Split top two windows into left/right halves |
| ⌃🌐⇧ ↑ | Split top two windows into top/bottom halves |
| ⌃🌐⇧ ↓ | Split top two windows into top/bottom halves |

---

## Three-Way Splits (add Option + Shift)

Tiles **three windows**: the front window takes one half, the next two fill the other half in quarters.

| Shortcut | Action |
|----------|--------|
| ⌃🌐⌥⇧ ← | Primary window on left, two quarters on right |
| ⌃🌐⌥⇧ → | Primary window on right, two quarters on left |
| ⌃🌐⌥⇧ ↑ | Primary window on top, two quarters on bottom |
| ⌃🌐⌥⇧ ↓ | Primary window on bottom, two quarters on top |

---

## Customizing Shortcuts

**System Settings → Keyboard → Keyboard Shortcuts → Windows**
- Assign custom shortcuts for Fill, Center, Halves, Quarters, and Arrange commands

---

## Title Bar Double-Click

- Default action: **Zoom** (resize to fit content)
- Change to **Fill** in: **System Settings → Desktop & Dock → Title Bar Double Click Action**

---

## Drag-to-Tile

Enable in **System Settings → Desktop & Dock → Windows**:

| Action | Result |
|--------|--------|
| Drag to left/right edge | Tile to that half |
| Drag to top edge | Fill the screen |
| Hold ⌥ while dragging | Easier tiling + quarter splits at corners |

> Toggle **"Tiled windows have margins"** off to remove gaps between tiled windows.
