# Mac Keyboard Shortcuts

> Reference based on [Apple Support article 102650](https://support.apple.com/en-us/102650)

**Modifier Key Symbols:** `Cmd (⌘)` | `Option/Alt (⌥)` | `Shift (⇧)` | `Control (⌃)` | `Fn`

---

## Cut, Copy, Paste & General

| Shortcut | Action |
|----------|--------|
| ⌘X | Cut selected item to clipboard |
| ⌘C | Copy selected item to clipboard |
| ⌘V | Paste clipboard contents |
| ⌘Z | Undo previous action |
| ⇧⌘Z | Redo (undo the undo) |
| ⌘A | Select all items |
| ⌘F | Open Find dialog |
| ⌘G | Find next occurrence |
| ⇧⌘G | Find previous occurrence |
| ⌘H | Hide front app windows |
| ⌥⌘H | Hide all other app windows |
| ⌘M | Minimize front window to Dock |
| ⌥⌘M | Minimize all windows of front app |
| ⌘O | Open selected item or file dialog |
| ⌘P | Print current document |
| ⌘S | Save current document |
| ⇧⌘S | Save As / duplicate |
| ⌘T | Open new tab |
| ⌘W | Close front window |
| ⌥⌘W | Close all windows of app |
| ⌘Q | Quit app |
| ⌥⌘Esc | Force quit an app |
| ⌘Space | Open Spotlight search |
| ⌃⌘Space | Open Emoji & Symbols viewer |
| ⌃⌘F | Toggle full-screen mode |
| ⌘, (comma) | Open front app preferences |
| ⌘` (backtick) | Switch between windows of front app |
| ⇧⌘` | Switch to previous window of front app |
| ⌥⌘D | Show/hide Dock |
| ⌃⌘Q | Lock screen |
| ⇧⌘Q | Log out (with confirmation) |
| ⌥⇧⌘Q | Log out immediately |

## Document Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘B | Bold selected text |
| ⌘I | Italicize selected text |
| ⌘U | Underline selected text |
| ⌘K | Add web link |
| ⌘; | Find misspelled words |
| Fn↑ | Page Up |
| Fn↓ | Page Down |
| Fn← | Scroll to beginning (Home) |
| Fn→ | Scroll to end (End) |
| ⌘↑ | Move cursor to beginning of document |
| ⌘↓ | Move cursor to end of document |
| ⌘← | Move cursor to beginning of line |
| ⌘→ | Move cursor to end of line |
| ⌥← | Move cursor to beginning of previous word |
| ⌥→ | Move cursor to end of next word |
| ⇧ + any above | Extend selection in that direction |
| ⌥Delete | Delete word to the left of cursor |
| FnDelete | Forward delete (delete right) |
| ⌃K | Delete from cursor to end of line |
| ⌃Y | Yank back deleted text |
| ⌥⇧⌘V | Paste and Match Style |
| ⌥Esc (or F5) | Auto-complete word |
